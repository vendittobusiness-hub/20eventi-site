## Cosa modificare
- Sostituisci il numero WhatsApp `393401234567` con il tuo (senza + e spazi).
- Sostituisci l'endpoint Formspree `https://formspree.io/f/xxxxxxxx` con quello reale.

## Deploy
1. Apri il repo su GitHub.
2. Incolla il contenuto di questi file nei rispettivi file del tuo progetto.
3. Commit → Vercel aggiorna il sito in automatico.
